HTML5-Notifications-and-Socket.io
=================================


Aplicação de exemplo explicada no vídeo: [http://youtu.be/HwSamlKBDn4](http://youtu.be/HwSamlKBDn4)


### Requerimentos
* NodeJS
* Socket.io

### Instalação
* Clone o repositório
* Entre na pasta e execute: `npm install`
* Inicie o server com `node app`
